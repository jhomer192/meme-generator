# Meme Generator

A clean, fast meme editor that runs entirely in the browser — no uploads, no watermarks.

**[Try it live →](https://jhomer192.github.io/meme-generator/)**

## Features

- **Template library** — common meme formats ready to go
- **Custom images** — upload any image from your device
- **Text editing** — top/bottom caption with font size, color, stroke, and shadow controls
- **Canvas export** — download the finished meme as a PNG
- **Dark/light themes** — multiple color themes

## Tech

React + TypeScript + Vite + HTML Canvas. Deployed to GitHub Pages.
